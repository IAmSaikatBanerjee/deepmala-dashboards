⚠️ Suno rules — read once (this is what makes or breaks the audio)

  • STYLE box = ALL the sound direction (instruments, mood, bpm, "no fade out"). LYRICS box = words ONLY.
  • Never put instrument/mood tags like [tanpura drone] in the Lyrics box — Suno will literally sing them as gibberish. Only structure tags [Chorus] [Verse] [Instrumental] are allowed there.
  • No dots or "……" — they cause garbled stretched vocals.
  • Custom Mode · Instrumental OFF (except the flute card) · model v4.5.

▶ How to use this sheet

  • Pick a card (ranked by watch-hour ROI). Copy STYLE → Style box, LYRICS → Lyrics box. That's it — already formatted correctly.

🎧 Make it addictive (watch-hour tricks)

  • "No fade out" is in every Style so the ending stays full-volume — that's what makes the Filmora loop join invisible.
  • Steady low BPM (60–74; sleep 50) — hypnotic, non-jarring, easy to leave on.
  • One mantra, repeated — repetition is the hook. Don't mix mantras in one track.
  • Authentic instruments only (tanpura, tabla, harmonium, bansuri, veena, bells). No western drums/synths.
  • Use-case titling — "for sleep / study / work / meditation / puja" → people press play and walk away = max minutes/view.
  • 🕉️ Sound identity (baked into every Style): authentic Hindu Sanatan, North-Indian classical raga-based, soft sweet enchanting voice, pure Sanskrit diction, temple atmosphere. 🎬 Filmora loop workflow

  1. Import the Suno mp3. Trim to a clean bar (cut any intro silence / messy tail so the join is smooth).
  2. Copy the clip and paste end-to-end (or right-click → repeat) until the timeline hits 1 hour (8 hrs for sleep). Crossfade ~0.5s at each join if you hear a seam.
  3. Add your still image / slow zoom (Ken Burns) of the deity as the video track — one calm visual is enough.
  4. Export 1080p, 24–30fps. For a Short: duplicate, crop 9:16, trim to 30–45s of the strongest passage.
  5. Upload → title with keyword + "1 Hour" → paste SEO description + affiliate links → thumbnail → AI-disclosure YES.
